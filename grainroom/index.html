Grain Room: A Virtual Reality Granular Synthesizer
In this project, PAT BFA student Logan Hughes developed a novel 3D granular synthesizer using the Unity game engine and PureData. The system allows the user to spray grains into 3D space and excite them with different types of brushes that are mapped to different grain window types. The user can also create imprints of brushes in midair to achieve sustained granulations. The grains can be pushed forwards and backwards to have them interact with brush imprints or to clear them from the scene.

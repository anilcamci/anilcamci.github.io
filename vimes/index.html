VIMEs: Virtual Interfaces for Musical Expression
Unforeseen Collisions is a networked VR piece for two performers playing a Virtual Interface for Musical Expression (VIME). The performance involves the collaborative construction of a musical causality system in a virtual space shared between the performers over a network. In this space, performers place a range of resonant objects in the pathway of marbles falling from pipes at player-adjustable intervals. These resonant objects can be spawned, positioned, tuned, muted, and destroyed. While the performance stems from a predetermined plan for a structured improvisation, the minute variations in the configuration of the building blocks gives the system an inherently indeterminate behavior. As the performers construct the piece, the virtual physics of the system moves it in unforeseen directions.

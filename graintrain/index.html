GrainTrain: Multi-touch Granular Synthesizer
GrainTrain is an innovative multi-touch performance tool for real-time granular synthesis based on hand-drawn waveform paths. It is a cross-platform web application that can run on both desktop and mobile computers, including tablets and phones. Allowing the user the draw their own interface for granular synthesis, it enables ergonomic multi-touch interactions and new performance techniques, such as superimposition of waveforms, forking interaction paths and indeterminacy through complex drawings.
